# RENAOMBER

Website produk streetwear & collectibles dibangun dengan Vite + React + TypeScript.

Terinspirasi dari [GOODBADORELSE](https://goodbadorelse.com/) dengan desain editorial berbeda — palet cream/forest green, layout hero split, filter kategori, dan cart drawer.

## Fitur

- Grid produk dengan filter kategori
- Status produk (Available, Sale, Sold Out, Pre-Order)
- Keranjang belanja (slide-out drawer)
- Newsletter signup
- Store policies
- Responsif (mobile & desktop)

## Menjalankan

```bash
npm install
npm run dev
```

Buka http://localhost:5173

## Build

```bash
npm run build
npm run preview
```
